# Reproduced datasets aligned to `IMF paper.pdf`

This folder contains two different reproduction approaches:

## 1) WEO-based alignment (existing)

Constructed from `WEO.csv`:

- `ghana_imf_reproduction_dataset.csv`
- `ghana_imf_reproduction_dataset_wide.csv`

## 2) PDF-table transcription (new)

Constructed directly from the IMF paper table **without using repository CSV inputs**:

- `imf_paper_ghana_selected_indicators_2022_2028.csv`

The file above is generated by:

```bash
python scripts/reproduce_from_imf_paper_pdf.py
```

It reproduces values from the table:

- **"Ghana: Selected Economic and Financial Indicators, 2022-28"** in `IMF paper.pdf`.
